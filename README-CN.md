# Conflux Map

[English](README.md) | **简体中文**

Conflux Map 是一个面向 Minecraft 1.17.1 的 Fabric 客户端小地图 / 世界地图模组。
对任意服务端（原版或模组端）均可独立运行，不需要服务端组件。功能行为参考了主流
小地图模组，但所有代码均从零实现；第三方组件与归属说明见
[`THIRD_PARTY_NOTICES.md`](THIRD_PARTY_NOTICES.md)。

## 功能特性

- **小地图 HUD** —— 常驻屏幕角落的浮层，方形或圆形，4 个角落位置、4 档尺寸、
  4 档缩放，可选跟随视角旋转，坐标与生物群系信息行。
- **全屏世界地图** —— 可拖动缩放的地图，多分辨率瓦片，以光标为中心缩放，
  右键点击可直接创建路径点。
- **洞穴 / 下界 / 末地图层** —— 主世界地下模式自动检测（带滞回，边界处不会来回
  闪烁）；下界当前层 / 基岩顶层 / 手动 Y 切片三种模式；末地虚空背景渲染。
- **路径点与死亡点** —— 创建、编辑、上色、分组、显示/隐藏切换；死亡自动生成
  死亡点；主世界与下界之间正确的 8:1 坐标换算；小地图边缘方向指示器提示
  范围外的路径点。
- **实体雷达** —— 敌对 / 被动 / 玩家 / 其他四类实体独立开关，各自可配置范围与
  数量上限。
- **磁盘缓存** —— 已探索地形按世界 / 服务器 / 维度 / 图层分别持久化，重新进入
  世界时直接显示此前画过的地图，而不是一片空白。
- **种子预测底图** —— 把地图拖到未探索的主世界或末地区域时，会立即显示基于种子
  推算的生物群系、地形高度和 1.17 树木候选，真实已探索的瓦片会在其上层覆盖。
  底层是内置的 [cubiomes](https://github.com/Cubitect/cubiomes) 原生库。
- **结构候选标记** —— 村庄、海底神殿、林地府邸、前哨站、废弃传送门、末地城会以
  半透明标记显示，直到真实数据确认或排除。
- **预测模式** —— 按 `P` 切换：*全部*（默认）、*仅已生成*（只在服务端已生成的
  区块上显示底图）、*仅已访问*（纯已捕获地图，不做预测）。
- **可选服务端 companion** —— 同一个 jar 可作为服务端 companion 运行，将预测与
  真实世界对比后返回紧凑的按列校正；只有在服务端显式开启时才会共享种子。
- **设置界面** —— 以上所有设置项均可在游戏内直接调整，改动立即生效、无需重启。
  完整的中英文本地化。

## 按键绑定

以下按键均可在 Minecraft 原版的按键设置界面中重新绑定，分类名为 "Conflux Map"。

| 默认按键 | 功能 |
|---|---|
| `H` | 开关小地图 |
| `]` / `[` | 小地图放大 / 缩小 |
| `M` | 打开全屏世界地图 |
| `Y` | 切换手动图层模式 |
| `U` | 打开路径点列表 |
| `B` | 在当前位置新建路径点 |
| `,` | 打开设置界面 |
| `P` | 切换预测模式（全部 / 仅已生成 / 仅已访问） |
| `F9` | 重载预测瓦片 |

## 构建

需要与 Minecraft 1.17.1 工具链兼容的 JDK（Java 16）；Minecraft、映射表、
Fabric API 由 Loom 自动下载。

```sh
./gradlew :1.17.1:build
```

构建产物输出到 `versions/1.17.1/build/libs/`。
